# healthcare_appointments_noshow_analysis

# Overview
Analysis of 110K medical appointment records from public health clinics in Vitoria, Brazil to identify no-show patterns using MySQL, Python, and Tableau

# Tools used
- SQL (data cleaning and preparation)
- Python (exploratory data analysis)
- Tableau (data visualization and reporting)
- MS Powerpoint (presentation preparation)

# Key findings
- Teenagers have the highest no-show rate at 26.64%
- Longer waiting times significantly increase no-show rates
- Scholarship (Bolsa Família) patients miss more appointments
- Santos Dumont neighbourhood has the highest no-show rate
